# Спецификация API

### Любому пользователю должен быть доступен просмотр аятов с пагинацией

Пример запроса:

```bash
curl -X 'GET' \
  'https://quranbot.ilaletdinov.ru/api/v1/ayats/?page_num=1&page_size=50' \
  -H 'accept: application/json'
```
